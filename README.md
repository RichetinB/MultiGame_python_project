# Jeu Multi Simple

Ce jeu multi simple est une application Python qui permet aux joueurs de s'affronter dans une partie compétitive en utilisant des classes et des améliorations.

## Fonctionnalités

- **Classes de personnages** : Choisissez parmi différentes classes de personnages, chacune avec ses propres caractéristiques et compétences uniques.
- **Améliorations** : Gagnez des points d'expérience et débloquez des améliorations pour renforcer vos compétences et capacités.
- **Partie compétitive** : Affrontez d'autres joueurs dans une partie compétitive pour déterminer le meilleur joueur.

## Comment jouer

1. **Installation**
   - Assurez-vous d'avoir Python installé sur votre système.
   - Téléchargez ou clonez ce dépôt sur votre machine.

2. **Exécution**
   - Ouvrez une console ou un terminal.
   - Accédez au répertoire du jeu (`cd MultiGame_python_project`).
   - Exécutez le fichier principal (`python main.py`).

3. **Choix de classe**
   - Choisissez une classe de personnage parmi les options proposées.
   - Chaque classe a ses propres statistiques et compétences uniques.

4. **Améliorations**
   - Jouez et gagnez des points d'expérience en remportant des parties.
   - Utilisez les points d'expérience pour débloquer et améliorer les compétences de votre personnage.

5. **Partie compétitive**
   - Affrontez d'autres joueurs dans des matchs compétitifs.
   - Gagnez des récompenses et montez dans le classement.
